# 创意工坊功能改造计划

## 需求概述
将创意工坊从"必须上传图片"改为"图片上传可选"，同时去除四个风格选择按钮，实现：
1. **文生图**：用户只输入文字描述 → 生成新图片
2. **图生图**：用户上传图片 + 文字描述 → 对图片进行转换
3. **去除风格选择**：不再需要选择苏绣/青花瓷/皮影戏/剪纸

## 实施步骤

### 步骤1：创建图片上传API

#### 1.1 新建 `backend/app/api/upload.py`
```python
from fastapi import APIRouter, UploadFile, File
from fastapi.responses import FileResponse
import os
import uuid
from pathlib import Path

router = APIRouter()

UPLOAD_DIR = Path("backend/uploads")
UPLOAD_DIR.mkdir(exist_ok=True)

@router.post("/upload-image")
async def upload_image(file: UploadFile = File(...)):
    """上传图片并返回URL"""
    # 生成唯一文件名
    ext = file.filename.split(".")[-1] if "." in file.filename else "png"
    filename = f"{uuid.uuid4()}.{ext}"
    filepath = UPLOAD_DIR / filename
    
    # 保存文件
    content = await file.read()
    with open(filepath, "wb") as f:
        f.write(content)
    
    # 返回URL
    return {
        "status": "success",
        "image_url": f"http://localhost:8002/uploads/{filename}"
    }
```

#### 1.2 在 `backend/app/main.py` 添加路由
```python
# 添加上传路由
from app.api.upload import router as upload_router

app.include_router(upload_router, prefix="/api/v1", tags=["upload"])

# 添加静态文件服务
from fastapi.staticfiles import StaticFiles
app.mount("/uploads", StaticFiles(directory="backend/uploads"), name="uploads")
```

### 步骤3：修改后端API端点

#### 3.1 修改 `backend/app/api/endpoints/creative_artisan.py`
- 将 `/image-to-image` 改为：
  - 参数 `image_url`: 可选（如果有图片）
  - 参数 `prompt`: 可选（如果有描述）
  - 逻辑：如果有image_url则图生图，否则文生图

```python
@router.post("/create-artwork")
async def create_artwork(
    image_url: Optional[str] = Form(None),
    prompt: str = Form("")
):
    """创建艺术品 - 支持文生图或图生图"""
    if image_url:
        # 图生图模式
        result = await cloud_image_service.image_to_image_from_url(
            image_url=image_url,
            prompt=prompt
        )
    else:
        # 文生图模式
        result = await cloud_image_service.generate_image(prompt)
    
    return {"status": "success", "image_url": result}
```
