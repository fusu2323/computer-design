# Phase 2 — UI Design Contract

> Visual and interaction contract for Phase 2: 小程序基础架构 (Mini-Program Foundation). Generated by gsd-ui-researcher.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (WeChat native + Taro 4.x) |
| Preset | not applicable |
| Component library | Taro (compiles to WXML/WXSS) |
| Icon library | wechat-ui (icon from @ant-design/wechat-ui or inline SVG) |
| Font | calligraphy (马善政/Ma Shan Zheng), xiaowei (站酷小薇体/ZCOOL XiaoWei), serif (思源宋体/Noto Serif SC) — loaded via @font-face in WXSS |

### Rationale

- Taro 4.x with React 19 is the cross-platform framework (D-01 in 02-CONTEXT.md)
- No shadcn/ui -- WeChat mini-program uses native WXML components (`view`, `text`, `image`, `button`, `navigator`)
- No framer-motion -- mini-program does not support web animation libraries; use Taro `useAnimation` hook or CSS `@keyframes` in WXSS
- Existing 6-color Eastern aesthetics palette from Phase 1 is preserved as WXSS custom properties
- Package size must stay under 20MB (main package under 15MB with lazy loading)

---

## Spacing Scale

8-point grid preserved from Phase 1, adapted for WXSS as CSS custom properties.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 8px | Icon gaps, tight inline padding |
| sm | 16px | Default element spacing, gap between card elements |
| md | 24px | Card padding, section internal spacing |
| lg | 32px | Section padding, layout gaps |
| xl | 48px | Major section breaks, hero spacing |
| 2xl | 64px | Page-level top/bottom spacing |

**WXSS declaration:**
```wxss
page {
  --space-xs: 8px;
  --space-sm: 16px;
  --space-md: 24px;
  --space-lg: 32px;
  --space-xl: 48px;
  --space-2xl: 64px;
}
```

**Exceptions:**
- Touch targets: minimum 44px (WeChat mini-program accessibility standard)
- WeChat login button height: 48px (larger than default for easy tap)
- Daily card hero image: full-width, aspect-ratio 16:9 (or 4:3 for portrait ICH images)

---

## Typography

3 sizes, 2 weights, adapted from Phase 1 for WXSS. Font files must be bundled in the mini-program package (use Taro's `static` folder or `copy` config).

| Role | Size | Weight | Line Height | Font Family | Usage |
|------|------|--------|-------------|-------------|-------|
| Body | 28rpx (~14px) | 400 (normal) | 1.6 | serif (Noto Serif SC) | ICH fact text, descriptions |
| Label | 24rpx (~12px) | 400 (normal) | 1.4 | xiaowei (ZCOOL XiaoWei) | Category tag, button labels |
| Heading | 36rpx (~18px) | 600 (semibold) | 1.3 | calligraphy (Ma Shan Zheng) | Card title, page title |
| Display | 48rpx (~24px) | 600 (semibold) | 1.2 | calligraphy (Ma Shan Zheng) | Hero title, app name |

**WXSS declaration:**
```wxss
page {
  --font-serif: 'Noto Serif SC', serif;
  --font-calligraphy: 'Ma Shan Zheng', cursive;
  --font-xiaowei: 'ZCOOL XiaoWei', serif;
  --font-size-body: 28rpx;
  --font-size-label: 24rpx;
  --font-size-heading: 36rpx;
  --font-size-display: 48rpx;
}
```

**Note on rpx vs px:** WeChat mini-program uses `rpx` (responsive pixel) for device-independent sizing. 1rpx = 0.5px on standard 750rpx-width devices. Use rpx throughout.

## Color

Phase 1 Eastern aesthetics palette, declared as WXSS custom properties. 60/30/10 ratio preserved.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #F7F5F0 (rice-paper) | Page background |
| Secondary (30%) | #FFFFFF (white) | Card background surfaces |
| Accent (10%) | #C04851 (vermilion) | Primary CTA button only |
| Accent reserved for | WeChat login button, primary card interaction | Never use for large fills or non-interactive decoration |
| Destructive | #C04851 (vermilion) | Not used in Phase 2 (no destructive actions) |

**Palette (adapted from Phase 1):**

| Token | Hex | Usage |
|-------|-----|-------|
| ink-black | #2B2B2B | Primary text, headings |
| rice-paper | #F7F5F0 | Page background, text on dark |
| vermilion | #C04851 | Primary CTA, accent highlights |
| cyan-glaze | #5796B3 | Category tag background (light tint: rgba(87,150,179,0.12)) |
| tea-green | #CCD4BF | Decorative accents, borders |
| charcoal | #4A4A4A | Secondary text, captions |
| white | #FFFFFF | Card backgrounds |

**WXSS declaration:**
```wxss
page {
  --color-ink-black: #2B2B2B;
  --color-rice-paper: #F7F5F0;
  --color-vermilion: #C04851;
  --color-cyan-glaze: #5796B3;
```

### Animations (CSS-only, no framer-motion)

```wxss
/* Card entrance: fade + slide up on page load */
@keyframes cardEnter {
  from {
    opacity: 0;
    transform: translateY(24rpx);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

.card-enter {
  animation: cardEnter 400ms ease-out forwards;
}

/* Login button press effect */
@keyframes buttonPress {
  0% { transform: scale(1); }
  50% { transform: scale(0.97); }
  100% { transform: scale(1); }
}

.login-btn:active {
  animation: buttonPress 100ms ease-out;
}

/* Loading shimmer for card skeleton */
@keyframes shimmer {
  0% { background-position: 200% 0; }
  100% { background-position: -200% 0; }
}

.skeleton {
  background: linear-gradient(90deg, #f0f0f0 25%, #e0e0e0 50%, #f0f0f0 75%);
  background-size: 200% 100%;
  animation: shimmer 1.5s infinite linear;
}
```

## Component Inventory

### DailyCard
- **States:** loading (shimmer skeleton), loaded (hero image + text), error (empty state)
- **Variants:** none (single variant for Phase 2)
- **Props:** imageUrl, category, title, factText
- **Touch target:** entire card is not tappable in Phase 2

### LoginButton
- **States:** default, pressed (scale 0.97), loading (spinner + disabled), disabled (after auth)
- **Variants:** primary (vermilion fill) only
- **Props:** onLoginSuccess callback
- **Behavior:** Triggers `wx.login()` on tap; shows loading spinner while processing

### CategoryTag
- **States:** default only
- **Appearance:** cyan-glaze pill, white text, xiaowei font
- **Props:** label (string)

### SkeletonCard
- **Purpose:** Loading placeholder while daily card data fetches
- **Appearance:** rounded rectangle with shimmer animation

### AppHeader
- **Purpose:** Shows app name "数字传承人" and subtitle
- **Appearance:** centered, calligraphy title, xiaowei subtitle
