## Files Modified

### `frontend/src/pages/MyPractice.jsx`

Refactored from hardcoded mock data to localStorage-driven:

**Removed:**
- SuEmbroidery and PurpleClay image imports
- Hardcoded stats: "12.5h", "85%", "3" (mastered skills), "15" (completed works), "Lv.2"
- Hardcoded project items (SuEmbroidery project, PurpleClay project)
- Hardcoded certificate with "李明"

**Added:**
- `useLocalStorage` hook import
- Reads from `vision_sessions` and `user_profile` localStorage keys
- Stats calculated dynamically:
  - `totalHours` — sum of session durations / 60
  - `avgScore` — mean of all session scores
  - `masteredSkills` — scenarios with 3+ sessions and avg >= 80
  - `completedWorks` — sessions with score >= 85
- Level derivation: Lv.1 (0-5 sessions), Lv.2 (6-15 or avg >= 60), Lv.3 (16+ with avg >= 80)
- Recent sessions display (last 5) with relative timestamps
- Empty state when no sessions exist
- Certificates from `user_profile.certificates`
