# PROJECT.md — 数字传承人 (The Digital Inheritor)

## 1. Concept & Vision

**数字传承人** 是一个面向中国大学生计算机设计大赛的 AI 驱动非遗 (ICH) 工艺教学与复原系统。

项目采用 **多智能体协作 (Multi-Agent) 架构**，由一个中央协调器统筹三个专业智能体，实现"能看、能听、会画"的数字化非遗导师体验。

**设计理念**: 东方美学与现代交互的融合 — 以数字之眼，观非遗之美。

## 2. Project Type

**参赛项目** — 中国大学生计算机设计大赛 · 软件应用与开发

## 3. Tech Stack

### Frontend
- React 19 + TypeScript + Vite

### Backend
